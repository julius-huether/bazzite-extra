# bazzite-extra
This repository is using the ublue template for custum bootc images (https://github.com/ublue-os/image-template/tree/main). I use this image to extend bazzite-dx-nvidia by the following functions:
- Reiner STC driver for Reiner chip-card readers (https://www.reiner-sct.com/chipkartenleser/)
- AusweisApp2 for German ID-Cards
- Thunderbird, Firefox and KeePassXC as RPM. This enables communication of KeePassXC plugin and Firefox.
